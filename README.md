# Celebal SQL Assignment

## Objective
The objective of this project is to analyze the Superstore sales dataset using SQL queries in DuckDB with Python and Jupyter Notebook.

## Tools Used
- Python
- Pandas
- DuckDB
- Jupyter Notebook

## Dataset
- Sample - Superstore.csv

## Tasks Performed
- Loaded the CSV dataset into DuckDB
- Explored the table schema and sample data
- Applied WHERE filters
- Performed GROUP BY aggregations
- Sorted and limited results
- Analyzed monthly sales trends
- Identified top customers
- Checked duplicate records
- Validated data quality
